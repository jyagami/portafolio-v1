Mis habilidades

Soy una persona autodidacta y dedico la mayor parte del tiempo a adquerir nuevos conocimientos y mezclarlos con las facultades desarrolladas en la Universidad. El diseño y desarrollo de aplicaciones móviles es mi habilidad mas desarrollada.

HTML & CSS

HTML , HyperText Markup Language, da estructura y significado al contenido al definir ese contenido como, por ejemplo, encabezados, párrafos o imágenes. CSS , u hojas de estilo en cascada, es un lenguaje de presentación creado para dar estilo a la apariencia del contenido, utilizando, por ejemplo, fuentes o colores.

Vanilla JavaScript

Vanilla JavaScript es un marco rápido, ligero y multiplataforma para crear aplicaciones JavaScript increíbles y potentes. Hoy en día, elegir vanilla JS vs jQuery es la opción más inteligente, por lo que su base de código no termina innecesariamente llena de dependencias.

PHP + Laravel

Laravel es un framework de código abierto para desarrollar aplicaciones y servicios web con PHP 5 y PHP 7. Su filosofía es desarrollar código PHP de forma elegante y simple, evitando el "código espagueti". Fue creado en 2011 y tiene una gran influencia de frameworks como Ruby on Rails, Sinatra y ASP.NET MVC.

ReactJS & React Native

ReactJS es una biblioteca para construir interfaces de usuario componibles. React Native es un marco de JavaScript para escribir aplicaciones móviles reales y nativas para iOS y Android.Se basa en React, la biblioteca JavaScript de Facebook para construir interfaces de usuario, pero en lugar de apuntar al navegador, apunta a plataformas móviles.

Flutter Dart

Flutter es un SDK de aplicaciones para crear aplicaciones de alto rendimiento y alta fidelidad para iOS, Android y web desde una única base de código y compildas nativamente. El objetivo es permitir a los desarrolladores ofrecer aplicaciones de alto rendimiento que se sientan naturales en diferentes plataformas.

GIT

Git es un software de control de versiones pensando en la eficiencia y la confiabilidad del mantenimiento de versiones de aplicaciones cuando éstas tienen un gran número de archivos de código fuente. Su propósito es llevar registro de los cambios en archivos de computadora y coordinar el trabajo que varias personas realizan sobre archivos compartidos.

Java Android Nativo

Java Android Nativo, Es el lenguaje nativo que usa Android. Cualquier aplicación que use directamente el hardware y se comunique con el sistema operativo, usará este código. Es un lenguaje muy extendido, presente en muchos dispositivos, tanto por la red como a nivel local y aprender a programar en él, nos permitirá crear aplicaciones para el sistema de Google, programas de escritorio, o incluso videojuegos.

Kotlin Android Nativo

Kotlin es un lenguaje de programación estáticamente tipado. Es un lenguaje que corre bajo la Máquina Virtual de Java , por lo que tiene el mismo rendimiento que Java, sin penalizar en ningún momento en este aspecto. Está diseñado para interoperar con Java, por lo tanto podemos tener módulos programados en Java y otros módulos desarrollados en Kotlin.

Python

Python es un lenguaje de programación interpretado de tipado dinámico cuya filosofía hace hincapié en una sintaxis que favorezca un código legible. Es un lenguaje de programación multiparadigma y disponible en varias plataformas. Python contiene una gran cantidad de librerías, tipos de datos y funciones incorporadas en el propio lenguaje, que ayudan a realizar muchas tareas comunes sin necesidad de tener que programarlas desde cero.
